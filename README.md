# 蜘蛛纸牌 `for Mac`

仿 `Windows` 经典单机游戏 `蜘蛛纸牌`

## 已完成

- [x] 基本游戏功能
- [x] 难易度设置
- [x] 分数、步数显示
- [x] 部分音频效果
